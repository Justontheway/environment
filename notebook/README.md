## notebook设置
* 主要用于python相关开发

### windows设置
#### 安装配置
* 安装python
* 在D盘创建对应目录结构
    * D:\Codebase
    * D:\Codebase\scripts
    * D:\Codebase\notebooks
    * D:\Codebase\notebooks\notebook2
* 将文件py2-notebook.bat放入D:\Codebase\scripts目录下
* 鼠标右键py2-notebook.bat -> 选择发送到 -> 桌面快捷方式
#### 使用说明
* 以后使用时只需要正常打开对应的桌面快捷方式就可以
* 新建的notebook文件都放到了D:\Codebase\notebooks\notebook2目录下
